
# Tharwa Backend

SOA-based e-banking system (REST), with an implementation of the OAuth authentication protocol. 

### Prerequisites
  - Oracle XE 11g
  - Nodejs v-8.0 
  - TypeScript 
  - Mocha 

### Installing

Avant de commencer executer la commande 

    - npm install
